# Car Service Logbook

A modern, responsive car service logbook application for tracking vehicle maintenance and service records.

## Features

- ✅ Track vehicle information (make, model, year, VIN)
- ✅ Record service details (date, type, mileage, cost, notes)
- ✅ View service history for each vehicle
- ✅ Clean, modern UI with automotive-themed design
- ✅ Local storage for data persistence
- ✅ Responsive design for all devices

## Tech Stack

- HTML5
- CSS3 (with modern features)
- JavaScript (ES6+)
- LocalStorage for data persistence

## Deployment

This application is ready to deploy on Vercel:

1. Fork this repository
2. Go to [Vercel](https://vercel.com) and sign up/in
3. Click "New Project" and import your forked repository
4. Vercel will automatically detect the settings and deploy your site
5. Your site will be live with a URL like `https://your-project.vercel.app`

## How to Use

1. Add your vehicles to the logbook
2. Record service details for each vehicle
3. Track maintenance schedules and expenses
4. View service history and statistics

## Local Development

To run locally:

```bash
npm install -g serve
serve -s .
```

Then visit `http://localhost:5000` in your browser.
